# Chat for Spotify

An iOS social app I designed, developed, and released in 2023. It used Spotify listening data to help people find others with similar music taste, explore music, and chat directly or in groups.

## Features

- Spotify sign-in, music profiles built from top artists, tracks, recent listening, and genres, plus track previews, likes, follows, and recommendations.
- User matching based on OpenAI embeddings of listening preferences and Pinecone similarity search.
- Firebase-backed direct and group chat with invitations, presence, unread counts, blocking, and reporting.
- AI-generated profile bios and a premium upgrade with verified StoreKit purchases.

## Stack

Swift · SwiftUI · UIKit · AVKit · StoreKit · Spotify Web API · Firebase Authentication · Realtime Database · Storage · Cloud Functions · OpenAI API · Pinecone · SDWebImageSwiftUI · UIImageColors · SwiftyChat
