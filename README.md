# Chat for Spotify

An iOS social app I designed, developed, and released in 2023. It used Spotify listening data to help people discover others with similar music taste, explore profiles and music, and talk through direct or group chats.

## Features

- Music-based user discovery and track recommendations, profiles, direct and group chat, presence, and invitations.
- Spotify sign-in and listening data, a Firebase real-time backend, AI-assisted matching and profile bios, and StoreKit 2 premium features.

## Stack

Swift · SwiftUI · UIKit · Spotify Web API · Firebase Authentication · Realtime Database · Cloud Functions · Storage · OpenAI API · Pinecone · StoreKit 2 · SpriteKit · AVFoundation
